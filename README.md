# E-Commerce Manual Testing Project
Manual Testing of E-Commerce 

## 📌 Project Overview
Manual Testing of a live E-Commerce Web & Mobile Application
covering end-to-end functional, regression, UI, and usability testing.

## 🔗 Application Under Test
- **Type:** E-Commerce Web & Mobile Application
- **Domain:** Retail / Online Shopping
- **Platform:** Web (Cross-Browser) + Mobile (Android)

## ✅ Testing Types Performed
- Functional Testing
- Regression Testing
- Smoke & Sanity Testing
- UI & Usability Testing
- Cross-Browser Testing
- Mobile App Testing
- Re-Testing & Bug Verification

- ## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| JIRA | Bug Tracking & Test Management |
| Excel | Test Cases & Bug Reports |
| BrowserStack | Cross-Browser & Mobile Testing |
| Figma | UI Validation against Design |
